# Weather App

A simple weather application that allows users to search for the current weather of any city using real-time data from the OpenWeather API. The app dynamically updates the user interface based on the response and provides clear feedback for invalid inputs or errors.

---

## Features

- Search weather by city name
- Displays:
  - City name
  - Temperature (Fahrenheit)
  - Humidity
  - Weather description
  - Weather-based emoji
- Error handling for:
  - Empty input
  - Invalid city names
  - Failed API requests
- Clean and responsive user interface

---

## How It Works

1. The user enters a city name and submits the form.
2. The app sends a request to the OpenWeather API.
3. Weather data is fetched asynchronously.
4. The page updates dynamically without reloading.
5. If an error occurs, an appropriate message is displayed.
